## 🛠️ Setup & Execution

### 1️⃣ Prerequisites
Ensure you have the latest **.NET SDK** installed.
- 📥 [Download .NET SDK](https://dotnet.microsoft.com/en-us/download)
- Verify installation:
  ```bash
  dotnet --version
  ```

### 2️⃣ Build and Run
Each program is grouped under its respective unit. To run a specific program, navigate to the root directory and use the `dotnet run` command with the `--project` flag.

```bash
# Build the entire solution/project
dotnet build

# Run a specific practical
dotnet run --project src/UNIT03/Program01
```
