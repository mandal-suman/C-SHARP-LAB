## Setup (.NET SDK)

### Install .NET SDK

1. Go to: https://dotnet.microsoft.com/en-us/download
2. Download **.NET SDK**
3. Install and verify:

```bash
dotnet --version
```

---

## Run

```bash
dotnet build
```

```bash
dotnet run --project src/UNIT03/Program01
```
